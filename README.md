## 🎁 Birthday Gift Site (Scrimba Exercise)

This project is a small **birthday‑themed webpage** built during a Scrimba HTML/CSS lesson.  
It includes a fun layout with a background image, a birthday message, and styled sections that recreate the look of a simple celebratory card.

The CodePen version used during the lesson is available here:  
https://codepen.io/Kernela/pen/xxvzZKz  

GitHub Pages may show slight visual differences compared to CodePen due to how images, fonts, and resource paths are handled, but the structure and purpose of the exercise remain the same.
